## Fragment Specification

Each `dist/fragments/<experiment_id>.html` is a self-contained HTML
snippet (no `<html>`, `<head>`, or `<body>` tags). It contains:

### Section 1: Overview
From `experiments.yaml`:
- Experiment number and name as `<h2>`
- Risk level as a `<span class="risk-badge risk-{level}">` where
  level is none | low | medium | high
- Enabled/disabled status as `<span class="exp-status">`
- Notes as `<p class="exp-notes">`

### Section 2: Configuration
From `config.yaml`:
- Description as `<p class="exp-description">`
- Temperature, max_tokens, repeat_penalty in a
  `<div class="exp-meta">` with individual `<span>` per value
- List of prompt files in a `<div class="prompt-list">`

### Section 3: Results
From `results/run_log.yaml` (if it exists):
- A `<table class="results-table">` with columns:
  Run | Model | Prompt | Status | tok/s | eval_count | Output
- Status cell: `<span class="status-badge {status}">{status}</span>`
- Output cell: `<a href="../results/{filename}">view</a>` if file exists,
  else `(file not found)`
- If no run_log.yaml exists: `<p class="no-runs">No runs yet.</p>`

## index.html Specification

The frontend developer will fill in the UI chassis via the
`generate_ui_chassis()` function you leave as a placeholder.

`index.html` must include:
1. `<link id="theme" rel="stylesheet" href="themes/e2b.css">` —
   the active theme stylesheet, swappable by the toggle button
2. htmx from CDN: `<script src="https://unpkg.com/htmx.org@1.9.10"></script>`
3. The sidebar experiment list — one button per experiment:
   ```html
   <button class="experiment-btn {enabled_class}"
           hx-get="fragments/{experiment_id}.html"
           hx-target="#detail-viewport"
           hx-swap="innerHTML">
     {number} · {name} · {RISK}
   </button>
   ```
   Where `enabled_class` is `enabled` or `disabled`.
4. The detail viewport: `<div id="detail-viewport">` with a
   placeholder message as default content.
5. The theme toggle button:
   ```html
   <button id="theme-toggle" onclick="toggleTheme()">
     Switch to e4b theme
   </button>
   ```
6. This JavaScript for the toggle (inline in index.html):
   ```javascript
   let currentTheme = 'e2b';
   function toggleTheme() {
     currentTheme = currentTheme === 'e2b' ? 'e4b' : 'e2b';
     document.getElementById('theme').href = `themes/${currentTheme}.css`;
     document.getElementById('theme-toggle').textContent =
       `Switch to ${currentTheme === 'e2b' ? 'e4b' : 'e2b'} theme`;
   }
   ```
