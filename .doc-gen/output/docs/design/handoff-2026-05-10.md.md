## Sr. Barbara — Staging Schema

Two new SQL files added to `sr-barbara/db/`:

- `schema_staging.sql` — creates `staging` schema mirroring production tables
  with added review metadata columns (model, prompt_label, corpus_label,
  run_id, review_status, reviewer_notes)
- `promote_staging.sql` — promotes approved rows from staging to public schema,
  marks them as promoted, resets sequences, runs verification query

Workflow: Gemma output → `staging` schema via Adminer → human reviews →
sets `review_status = 'approved'` → runs promote_staging.sql → production.
